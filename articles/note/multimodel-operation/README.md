## 次のステップ（試すフェーズ）

### 扉絵生成

```
1. browser tool で clawd プロファイルの Gemini を開く
2. プロンプト:
   "Minimalist digital illustration of an AI factory control room.
   Three distinct robot workers at different stations - one large silver robot supervising monitors,
   one medium beige robot processing documents, one small greyish-beige robot doing repetitive tasks.
   Color palette: silver, beige, greige tones. Clean geometric shapes, soft shadows.
   Professional tech aesthetic, no text."
3. 生成された画像をダウンロード → cover.png として保存
```

### Note自動アップ手順（参考：cooking-cypherpunk）

```
1. browser で note.com を開く（clawdプロファイル）
2. 新規記事作成ページへ
3. article.md の内容をペースト
4. cover画像をアップロード
5. ハッシュタグを設定
6. 公開
```
